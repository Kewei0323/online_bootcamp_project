About this Project

This project set out to extract a large amount of data from the copious Stack Overflow dataset, develop machine learning models to predict outcomes for questions and users, and enable people to challenge themselves to predict as well as a machine. To do this, the project relies upon a variety of technologies. These include:

  • Google BigQuery, a SQL-based data warehouse on Google Cloud Platform, which Stack Overflow uses to make its data available to the public;
  • Apache Spark, an open source big data framework
  • H2O, an open source machine learning platform
  • AWS Lambda serverless compute service
  • Python programming language for use with the PySpark and H2O APIs and for web development
  • Flask
  • Bootstrap
  • Ubuntu Linux
